Supervision in Algorithms Machine Learning algorithms can be grouped based on the way they learn. And in this grouping are the algorithms: Supervised, Non-Supervised, Semi-Supervised. Supervised An algorithm is supervised when it has input variables (x) and an output variable (y) and the algorithm learns the function that maps the input to the output. The objective here is to approximate the objective function in such a way that when new data x is entered, the result y can be predicted for that data. It is called supervised learning because the process in which an algorithm learns from training data can be likened to a teacher supervising the learning process. Most Machine Learning practices this type of learning. Supervised learning problems can be grouped in two main groups:
    Classification: When the output variable is a category (such as red or white, etc). Regression: When the output variable is a real value (such as dollars or peso, etc).
Some examples of algorithms of this type are:
    Linear regression for regression problems. Random forest for classification and regression problems. Support Vector for classification problems.
Non-Supervised Unlike supervised algorithms, unsupervised algorithms have no corresponding output, only the input data (x). The goal in this type of algorithm is to model the underlying structure or distribution of the data in order to learn more about it. Note: They are called Non-Supervised because there is no resemblance to the analogy of the teacher teaching, since there is no correct answer. Non-Supervised learning problems can be grouped into two main groups:
    Clustering: This is when you want to discover the inherent clustering of the data. Association: When you want to discover the rules that describe a large portion of the data.
Examples Some examples of such algorithms are: K-means for clustering problems. Apriori Algorithm for association problems. Semi-supervised As can be guessed, semi-supervised algorithms are a hybrid between supervised and unsupervised algorithms, being able to take advantage of both. These types of algorithms are ideal for when you have a large amount of data (x) as input and only a little of the data is categorized (y). Many real-world Machine Learning problems fall into this category. Examples
    Label Propagation Semi-supervised SVMs
Summary Algorithms can be grouped in the way they learn, where:
    Supervised: All data is categorized, and the algorithms learn to predict the outcome of the input data. Unsupervised: All data is uncategorized, and the algorithms learn the inherent structure of the data. Semi-Supervised: Some of the data is categorized, but most of the data is not, a mix between supervised and unsupervised. Techniques from both categories can be used.
